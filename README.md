Smart Contract Developer
------------------------

I am a blockchain and full-stack developer focused on building decentralized applications and modern web systems. I specialize in smart contract development with Solidity and have hands-on experience designing and implementing token standards such as Fungible Tokens (FT), Non-Fungible Tokens (NFT), and DeFi protocols.

My technical stack includes JavaScript, React, Next.js, HTML, CSS, Tailwind CSS, Node.js, and Web3 tools such as Viem, Wagmi, RainbowKit, Foundry, Remix, and IPFS. I also work with SQL and MongoDB for data management and have programming experience in Java and Kotlin.

I am particularly interested in decentralized finance, protocol design, and blockchain infrastructure, and I actively contribute to open-source projects while continuously improving my skills in building secure, scalable, and user-friendly Web3 applications.
